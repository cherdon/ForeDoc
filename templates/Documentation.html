    Fore Documentation (1.0.1)


    The Fore Documentation helps to guide you on how to use the simulation program, inclusive of world generation and scenario simulation.
    You may use it as a reference, or simply read it from the start to the end.

    If this is the first time you are using the simulation program, take a look at the introductory tutorials at Tutorials Page, and see the Setting Up Instructions.
