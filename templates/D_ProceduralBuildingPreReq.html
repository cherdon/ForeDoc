    Pre-Requisites

    The procedural building feature is built with CAD software, and plugins to assist with the quick generation of textures and shapes.

    Before you begin, ensure that you have the following installed:

    Software
    • Rhinoceros 6 (5 is also okay)
    Plugins
    • Grasshopper (if you are using Rhinoceros 5)
    • Elk 2.2.2
    • Human
      If you are using Rhino 6, download Human for Rhino 6:


      If you are using Rhino 5, download Human + Tree Frog:

    Note: Please ensure that you know which version of Rhinoceros you are using as it will determine which version of the recommended plugins you need to download. Please refer to the installation instructions that are provided within each download if you are unsure on how to install them
